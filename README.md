# Dashboard-Bi
This project is a Power BI dashboard designed to analyze chocolate sales data and generate meaningful business insights.

The dashboard provides a comprehensive view of sales performance, customer behavior, and product trends, helping users understand key factors driving revenue.

Key Features:
- KPI metrics such as Total Sales, Total Orders, and Average Order Value
- Sales analysis by chocolate category and product type
- Time-based sales trends to track performance over different periods
- Top-performing products and customers
- Interactive filters for dynamic data exploration
- Clean and visually appealing dashboard layout

The goal of this project is to transform raw chocolate sales data into actionable insights using data visualization and business intelligence techniques.

This project demonstrates skills in:
- Data visualization using Power BI
- Data analysis and storytelling
- Building interactive dashboards
- Extracting business insights from datasets
